## How to donwload?
run this commands on terminal
```
cd Desktop
git clone https://github.com/theiskaa/whatsapp_redesing
```
then you have to run pub get command
```
flutter pub get
```
And then you can run the application
```
cd whatsapp_redesing 
flutter run
```
